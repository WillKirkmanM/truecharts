**`floats`**
Accepted formats are:

- `1.0000`
- `10.0000`
- `100.0000`
- `1000.0000`
- etc

Regex used to match this: `^\d{1,}\.[0-9]{6}$`
You can try live [here](https://regex101.com/r/BTJv6n/1)
